# 🌳 El bosque de las figuras literarias

Este recurso didáctico digital está diseñado para ayudar a estudiantes a explorar y practicar el uso de **figuras literarias** de forma interactiva, creativa y lúdica.

## 🎯 Objetivo del recurso
Desarrollar habilidades para **identificar**, **comprender** y **crear** expresiones literarias usando figuras como metáfora, símil, personificación e hipérbole.

## 📂 Contenidos

- **Portada interactiva:** Introduce el tema con un botón de inicio.
- **Actividad 1 – “Completa con creatividad”:** El estudiante completa frases usando una figura literaria.
- **Actividad 2 – “Cazador de figuras”:** Se seleccionan las figuras presentes en un fragmento narrativo.
- **Actividad 3 – “Mi frase creativa”:** El estudiante recibe palabras aleatorias y crea una frase literaria.
- **Evaluación formativa – “Caja de ideas locas”:** Retroalimentación inmediata sobre una frase libre.
- **Evaluación sumativa – “Mi mini cuento con estilo”:** Producción de un texto breve con uso de figuras literarias y autoevaluación mediante rúbrica.

## 🛠 Archivos incluidos

- `index.html`: Estructura y contenido del recurso.
- `style.css`: Estilos visuales.
- `script.js`: Lógica e interactividad.

## 🚀 Cómo usar

1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador.
3. Explora y utiliza cada actividad en orden.
4. Opcional: Publica este recurso con [GitHub Pages](https://pages.github.com/) para compartirlo fácilmente.

## 👩‍🏫 Ideal para

- Docentes de lengua y literatura.
- Estudiantes de secundaria.
- Actividades en clase o tareas autónomas.
